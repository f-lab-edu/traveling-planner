<h1>traveling-planner</h1>

해외나 국내 여행시에 여행 계획을 세우고 관리하는 서비스이다.<br>
서버관리 위주로 프로젝트를 진행하기 위해서 카카오 오븐을 통해서 프론트 프로토 타입을 작성했다. <br>
카카오 오븐 URL : https://ovenapp.io/view/hj5M5f8kvWAOS93N65BCPOXTvUJ7GjeT/xCORz

<br>
<h2>프로젝트 사용 기술</h2>

- Spring Boot
- Java 8
- MySQL
- Intellij IDEA
